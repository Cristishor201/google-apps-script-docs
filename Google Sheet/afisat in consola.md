# afisat - in consola

### afisat text simplu

```js
Logger.log("some text here");
```

### afisat map

Pentru vazut continutul unui hashmap:

```js
Logger.log( JSON.stringify( Object.fromEntries( myMap) ) );
```

### string formating

```js
Logger.log(`The total is ${10 + 25}`); // The total is 35
```

> [!NOTE]
> Nu se folosesc `"` ghilimele simple sau duble, ci <code>`</code> backtik
